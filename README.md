 Grocery-Delivery-Platform
A platform that enables, Product  listing, Cart and Delivery tracking 
Name: Oya Deliver

A modern grocery delivery web app built with React, designed and developed as a university group project.

 Live Demo Deployed on Vercel
[View App](https://grocery-delivery-platform-eta.vercel.app)


 Features
 Home page with animated search bar and category browsing
 Search and filter products by category
 Add to cart with live cart count
 Product detail page with quantity selector
 Checkout and payment page
 Order tracking with delivery status bar
 Order history
 User profile with editable details
Login and Signup with password visibility toggle
404 page

 Built With
- React
- React Router DOM
- Context API (Cart & Auth)
- CSS
- Backend - Local Storage

```bash
cd Grocery-Delivery-Platform

npm start
```
